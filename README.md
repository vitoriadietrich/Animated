# Animated Login Form using HTML And CSS

Um projeto simples de **formulário de login com animações** usando HTML e CSS.

## ✨ Sobre

Este projeto demonstra como criar um **login form moderno com animações**, deixando a interface mais interativa e atraente.

## 🛠️ Tecnologias

- HTML
- CSS (animation, transition)

## 🎯 Funcionalidades

- Animações suaves nos inputs
- Efeitos de foco e hover
- Design moderno e responsivo

## 🚀 Como usar

1. Baixe ou clone o repositório  
2. Abra o arquivo `index.html` no navegador
